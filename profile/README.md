## Example Query

> "Is DDR1 expression higher in KIRP vs KIRC?"

BioQuery understands your question, fetches data from TCGA, runs statistical tests, generates visualizations, and provides a natural language answer with p-values and effect sizes.

## Data Sources

- **TCGA** - 33 cancer types, 11,000+ patients
- **TARGET** - 7 pediatric cancers
- **GTEx** - 54 normal tissue types
- **CCLE** - ~1,000 cancer cell lines
- **CPTAC** - Proteomics data
- **GENIE** - Real-world clinical data
